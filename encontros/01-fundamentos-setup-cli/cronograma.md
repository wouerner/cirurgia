# Cronograma Detalhado: Encontro 01 (2h 30min)

### 1. Conceito: A Mentalidade 10% (30 min)
- **Abertura:** O fim da era do IDEs tradicionais (apenas sugestões) e o início da era "Agentic" (execução autônoma).
- **Novo Papel:** Por que saber *ler* código e entender *arquitetura* tornou-se mais importante do que decorar sintaxe.
- **Segurança:** Demonstração do que acontece se você vazar uma chave de API e como o Gemini CLI ajuda a prevenir isso respeitando o `.gitignore`.

### 2. Live Coding: Setup e Primeiro Projeto (1h 15min)
- **Arsenal:** Instalação ao vivo do Node.js, Gemini CLI e configuração da API Key do Google AI Studio.
- **Comandos Essenciais:** Demonstração de como navegar em um projeto usando `read_file`, `grep_search` e `list_directory`.
- **Intenção vs. Código:** Criando um projeto do zero (ex: Validador de Documentos) apenas descrevendo requisitos para a IA via terminal.
- **Refining:** Como corrigir a IA quando ela comete erros durante a geração inicial.

### 3. Laboratório: Mãos na Massa (45 min)
- **Setup Pessoal:** Garantir que todos os alunos tenham o ambiente funcionando.
- **Desafio:** Criar uma API simples (Node.js ou Python) com 2 endpoints e seus respectivos testes unitários usando 100% de comandos via Gemini CLI.
- **Peer Review:** Alunos revisam o código gerado pelos colegas para identificar possíveis melhorias de Clean Code sugeridas pela IA.
