# Cronograma Detalhado: Automação de Workflows com Scripts de IA (2h 30min)

## 1. Conceito (30 min)
- **O Poder do Scripting:** Como ganhar tempo automatizando tarefas repetitivas via scripts em Python, Bash ou Node.js.
- **Manipulação de Dados:** Uso de IA para criar ferramentas CLI de transformação de JSON, CSV e logs.

## 2. Live Coding (1h 15min)
- Demonstração prática dos conceitos de Automação e Scripts.
- Resolução de problemas reais usando Gemini CLI.

## 3. Laboratório (45 min)
- **Criação de Script Utilitário:** Automatizar uma tarefa manual semanal (ex: limpeza de logs ou geração de reports).
- **Refinamento de Script:** Adição de logs de sucesso/erro e indicadores de progresso.
- Suporte e Peer Review.
