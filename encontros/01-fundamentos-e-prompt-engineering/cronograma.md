# Cronograma Detalhado: Encontro 01 (2h 30min)

### 1. Conceito: Mentalidade 10% e Prompt Engineering (30 min)
- **A Mentalidade Agentic:** Por que saber *ler* código e entender *arquitetura* tornou-se mais importante do que decorar sintaxe.
- **Técnicas de Elite:** System Prompting, Few-Shot e Chain-of-Thought aplicado a código complexo.
- **Segurança:** Como o Gemini CLI ajuda a prevenir vazamento de dados respeitando o `.gitignore`.

### 2. Live Coding: Setup, Arsenal e Workflow Agentic (1h 15min)
- **Arsenal:** Instalação ao vivo do Gemini CLI e configuração da API Key.
- **Comandos Essenciais:** Navegação em projetos usando `read_file`, `grep_search` e `list_directory`.
- **Planejamento:** Usando o `enter_plan_mode` para desenhar uma feature antes de codar.
- **Refining:** Como corrigir a IA e interagir com sub-agentes para refinar a implementação em tempo real.

### 3. Laboratório: Mãos na Massa (45 min)
- **Setup Pessoal:** Garantir que todos os alunos tenham o ambiente funcionando.
- **Desafio:** Criar uma API simples com 2 endpoints e seus respectivos testes unitários usando 100% de comandos via Gemini CLI e prompts estruturados.
- **Peer Review:** Alunos revisam o código gerado pelos colegas para identificar possíveis melhorias de Clean Code sugeridas pela IA.
