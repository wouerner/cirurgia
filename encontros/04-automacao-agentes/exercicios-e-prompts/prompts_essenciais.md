# Prompts Essenciais - Encontro 06

Foco em Automação Avançada, Agentes Autônomos e Scripts Customizados.

## 1. Delegação de Missão para Sub-Agente
```text
@generalist Localize todos os arquivos de configuração de ambiente, valide se as variáveis obrigatórias estão documentadas no `.env.example` e, se não estiverem, atualize o arquivo automaticamente.
```

## 2. Criação de Script de Automação de Changelog
```text
Crie um script em Python/Bash que utilize o Gemini CLI para analisar as mensagens de commit da última semana e gerar um `CHANGELOG.md` formatado por categorias (Features, Bug Fixes, Chores).
```

## 3. Prompt para Agente de Refatoração em Lote
```text
Atue como um agente de manutenção. Sua tarefa é percorrer todos os arquivos na pasta `src/components`, identificar componentes que não usam TypeScript estrito e convertê-los, garantindo que nenhum erro de tipagem seja introduzido.
```

## 4. Configuração de Agente Customizado (.gemini-policy)
```text
Me ajude a escrever uma política customizada para o Gemini CLI neste projeto, definindo que ele deve sempre usar [PADRÃO X] e nunca sugerir o uso da biblioteca [Y].
```
