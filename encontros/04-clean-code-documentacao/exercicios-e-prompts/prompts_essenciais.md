# Prompts Essenciais - Encontro 04

Foco em Refatoração (SOLID/Clean Code) e Documentação Técnica Automatizada.

## 1. Refatoração para SRP (Single Responsibility Principle)
```text
Analise a classe/arquivo [NOME] e identifique múltiplas responsabilidades. Proponha uma refatoração que divida o código em unidades menores, seguindo o Princípio de Responsabilidade Única (SRP).
```

## 2. Geração de Diagrama de Sequência (Mermaid)
```text
Com base no código de [ARQUIVO], gere um diagrama de sequência em Mermaid.js que ilustre o fluxo de dados desde a entrada do usuário até a resposta final do sistema.
```

## 3. Explicação de Código Legado
```text
Eu herdei este código e não entendo seu propósito: [COLE O CÓDIGO OU APONTE O ARQUIVO]. Explique em linguagem simples o que ele faz, quais são seus efeitos colaterais e sugira 3 melhorias de legibilidade.
```

## 4. Geração de README de Módulo
```text
Gere um arquivo `README.md` técnico para a pasta [CAMINHO]. Inclua uma descrição da arquitetura, as principais dependências e um exemplo de como utilizar as funções exportadas por este módulo.
```
