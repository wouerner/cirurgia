# Checklist do Showcase - Encontro 07

Este encontro é dedicado à apresentação dos workflows e automações criadas durante o curso.

## 1. Preparação da Apresentação
- [ ] Escolha o seu "melhor momento" (ex: uma refatoração complexa ou um agente customizado).
- [ ] Prepare um roteiro: Qual era o problema? Como o Gemini CLI ajudou? Qual foi o resultado final?
- [ ] Garanta que o código esteja limpo e o repositório organizado.

## 2. Prompts para o "Showcase"
Use estes prompts para gerar um resumo final da sua evolução.
```text
Analise as mudanças que fiz neste repositório durante o curso e gere um relatório de impacto, destacando as melhorias de arquitetura, testes e legibilidade que implementamos.
```

## 3. Ideias para o Projeto Final
- **Agente Especializado:** Um sub-agente configurado para o seu contexto de trabalho real.
- **Pipeline IA-First:** Um fluxo de CI/CD que usa IA para auditoria de segurança ou performance.
- **Refatoração de Legado:** A transformação completa de um módulo "impossível" em código moderno e testado.
