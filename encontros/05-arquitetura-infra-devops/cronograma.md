# Cronograma Detalhado: Encontro 05 (2h 30min)

### 1. Conceito: IA como Consultora de Sistemas (30 min)
- **Arquitetura Distribuída:** Validando decisões de mensageria, cache e banco de dados com a IA.
- **DevSecOps:** A mentalidade de segurança antecipada ("Shift-Left") usando escaneamento de vulnerabilidades via CLI.
- **IaC First:** Por que não criar recursos via console se a IA pode escrever o Terraform para você.

### 2. Live Coding: Do Docker ao Deploy (1h 15min)
- **Containerização:** Criando Dockerfiles e Docker Compose otimizados e seguros.
- **Infra as Code:** Geração de scripts Terraform para provisionamento de recursos em nuvem (AWS/GCP).
- **Security Audit:** Analisando dependências e o próprio código em busca de falhas de segurança conhecidas.

### 3. Laboratório: Arquiteto de Nuvem (45 min)
- **Desafio:** Desenhar a arquitetura de um sistema que suporte um aumento de 10% na carga e gerar todos os arquivos de configuração necessários (Docker, K8s ou Terraform).
- **Vulnerability Hunting:** Usar a IA para encontrar e propor correções para 3 problemas de segurança no código.
