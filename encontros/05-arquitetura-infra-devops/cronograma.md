# Cronograma Detalhado: Arquitetura, Diagramas e Trade-offs (2h 30min)

## 1. Conceito (30 min)
- **IA como Consultora Sênior:** Análise de trade-offs arquiteturais (ex: Redis vs Memória) e decisões de design.
- **Visualização de Infraestrutura:** Conversão de descrições textuais em diagramas Mermaid.js.

## 2. Live Coding (1h 15min)
- Demonstração prática dos conceitos de Arquitetura e Diagramas.
- Resolução de problemas reais usando Gemini CLI.

## 3. Laboratório (45 min)
- **Evolução de Arquitetura:** Proposta de mudanças para suportar escala de 10x usuários.
- **Solidificação Visual:** Geração de diagramas de arquitetura para propostas de melhoria.
- Suporte e Peer Review.

# Cronograma Detalhado: IA para DevSecOps e Infraestrutura (IAC) (2h 30min)

## 1. Conceito (30 min)
- **Infraestrutura como Código (IaC) com IA:** Criação de manifestos K8s, Dockerfiles e scripts Terraform.
- **IA para Segurança (SAST):** Identificação de chaves expostas e vulnerabilidades em imagens Docker.

## 2. Live Coding (1h 15min)
- Demonstração prática dos conceitos de DevSecOps e Infraestrutura (IaC).
- Resolução de problemas reais usando Gemini CLI.

## 3. Laboratório (45 min)
- **Orquestração com Docker Compose:** Criação de ambientes completos com banco de dados e cache.
- **Auditoria de Segurança:** Revisão de código em busca de vulnerabilidades como Injeção de SQL.
- Suporte e Peer Review.
